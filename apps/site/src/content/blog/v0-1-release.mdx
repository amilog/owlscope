The first public release is out. Highlights:

- **JavaScript SDK** — `owlscope/auto` captures console, fetch, XHR, errors.
- **Flutter SDK** — `owlscopeAuto()` captures print, http, FlutterError,
  navigator, frame timings.
- **Riverpod adapter** — `owlscope_riverpod` 0.1 with side-by-side state diff.
- **Desktop app** — Logs / Network / State / Errors / Performance / Timeline.
- **adb reverse helper** — `dart run owlscope:reverse` for physical Android.

Install via npm or pub.dev, grab the desktop app from
[Download](/download).
