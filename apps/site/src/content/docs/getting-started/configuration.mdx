OwlScope offers three configuration levels. Pick the one that matches how much
control you want.

## Level 1 — Zero config

The simplest possible setup. Auto-detects everything, sensible defaults.

<Tabs>
  <TabItem label="JavaScript">
    ```ts
    import 'owlscope/auto';
    ```
  </TabItem>
  <TabItem label="Flutter">
    ```dart
    import 'package:owlscope/auto.dart';
    
    void main() => owlscopeAuto(() => runApp(const MyApp()));
    ```
  </TabItem>
</Tabs>

Use this when you just want everything captured with no fuss.

## Level 2 — Config file / arguments

Override port, name, plugins, redaction.

<Tabs>
  <TabItem label="JavaScript">
    ```ts title="owlscope.config.ts"
    export default {
      name: 'web-app',
      host: 'localhost',
      port: 9090,
      redact: ['password', 'token', 'authorization'],
    };
    ```
  </TabItem>
  <TabItem label="Flutter">
    ```dart
    owlscopeAuto(
      () => runApp(const MyApp()),
      name: 'mobile-app',
      host: 'localhost',
      port: 9090,
      httpMaxBodyBytes: 16 * 1024 * 1024,
    );
    ```
  </TabItem>
</Tabs>

## Level 3 — Manual API

Full control over plugins, transports, redaction, custom events.

<Tabs>
  <TabItem label="JavaScript">
    ```ts
    import { createClient, ConsolePlugin, NetworkPlugin } from 'owlscope';
    
    const client = createClient({ name: 'my-app', port: 9090 });
    client.use(new ConsolePlugin());
    client.use(new NetworkPlugin({ ignore: [/\/health$/] }));
    client.start();
    ```
  </TabItem>
  <TabItem label="Flutter">
    ```dart
    import 'package:owlscope/owlscope.dart';
    
    OwlScope.configure(name: 'my-app', port: 9090)
      ..use(HttpPlugin(maxBodyBytes: 0)) // unlimited
      ..use(ErrorPlugin())
      ..use(NavigationPlugin(owlscopeNavigatorObserver));
    ```
  </TabItem>
</Tabs>

Use this when you need custom plugins, custom transports, or fine-grained
control over what's captured.

## Common options

| Option | Default | What |
|---|---|---|
| `name` | hostname | Client name shown in the desktop sidebar |
| `host` | `localhost` | OwlScope server host |
| `port` | `9090` | OwlScope server port |
| `redact` | `[]` | Keys to scrub from payloads |
| `silent` | `true` | Suppress SDK's own console output |
