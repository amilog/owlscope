This guide gets you from a fresh install to seeing your first event in under
three minutes.

## 1. Add OwlScope to your app

If you haven't already, follow the [installation guide](/getting-started/installation/)
to add the SDK and download the desktop app.

## 2. Open the desktop app

Launch OwlScope. You should see the Home screen with a "waiting for clients"
status. The server is listening on `localhost:9090` by default.

## 3. Trigger an event

Run your app and call:

```ts
console.log('hello from owlscope');
```

Or in Flutter:

```dart
print('hello from owlscope');
```

## 4. See it in the desktop UI

Switch to OwlScope. You'll see the event appear in the **Logs** panel within
a few hundred milliseconds.

## 5. Try a network call

Make any `fetch()` call (or `http.get()` in Flutter) — it'll show up in the
**Network** panel with full request/response inspection, headers, body, and
timing.
