OwlScope is a desktop debugger built for **mobile developers**. It captures
everything happening inside your React Native or Flutter app — console
output, network calls, state changes, errors, frame timings — and streams
it to a desktop UI you actually want to look at.

First-class support for both React Native and Flutter, in one tool.

## What you get

- One SDK per platform — `owlscope` on npm, `owlscope` on pub.dev
- One desktop app — macOS / Windows / Linux
- One `setup` command — fixes Info.plist + adb reverse so every target
  works without per-device tweaking
- One line of code — `startOwlScope()` (RN) or `owlscopeAuto()` (Flutter)
- Production-safe — no-ops in release builds

## What you don't get

- A SaaS dashboard. Your data never leaves your machine.
- A subscription. OwlScope is free and MIT licensed.
- Vendor lock-in. The wire protocol is documented; you can write your
  own UI.
