OwlScope is a desktop debugger that captures everything happening inside your
JavaScript or Flutter app — console output, network calls, state changes,
errors, frame timings — and streams it to a UI you actually want to look at.

It's the love child of Reactotron, Chrome DevTools, and Redux DevTools, with
none of the per-framework lock-in.

## What you get

- A **single SDK** to install (`owlscope` on npm, `owlscope` on pub.dev)
- A **single desktop app** (macOS / Windows / Linux)
- **Zero config** entry: `import 'owlscope/auto'` and you're done
- **Auto-detection** for Redux, Riverpod, Bloc, Axios, Dio, and more
- **Production-safe** — no-ops in release builds, no leaked tokens

## What you don't get

- A SaaS dashboard. Your data never leaves your machine.
- A subscription. OwlScope is free and MIT licensed.
- A vendor lock-in. The wire protocol is documented; you can write your own UI.
