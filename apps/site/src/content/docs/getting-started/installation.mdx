OwlScope has two pieces:

1. **The SDK** — a tiny library that runs inside your app
2. **The desktop app** — a UI that displays everything the SDK captures

Install both, connect them, and you're debugging.

## Prerequisites

- **JavaScript / TypeScript:** Node.js 18 or newer
- **Flutter:** Dart SDK 3.0 or newer
- **Desktop app:** macOS 10.15+, Windows 10+, or Linux with glibc 2.28+

## Step 1 — Install the SDK

<Tabs syncKey="js-pkg">
  <TabItem label="npm">
    ```bash
    npm install -D owlscope
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add -D owlscope
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add -D owlscope
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add -d owlscope
    ```
  </TabItem>
</Tabs>

For Flutter, add to your `pubspec.yaml`:

```yaml
dev_dependencies:
  owlscope: ^0.1.0
```

Then run `flutter pub get`.

## Step 2 — Download the desktop app

<a href="/download" class="action primary" style="display: inline-flex; align-items: center; gap: 8px;">

:::note[First-time install on macOS?]
Since OwlScope is open source, we don't pay Apple's $99/yr fee, so macOS will
warn you on first launch. Right-click → Open → Open in the dialog. After that
you'll never see it again. [Full guide →](/download)
:::

## Step 3 — Wire them up

Add a single import to your app's entry point:

<Tabs syncKey="framework">
  <TabItem label="React">
    ```ts title="src/main.tsx"
    import 'owlscope/auto';

    import { createRoot } from 'react-dom/client';
    import App from './App';

    createRoot(document.getElementById('root')!).render(<App />);
    ```
  </TabItem>
  <TabItem label="React Native">
    ```ts title="index.js"
    import 'owlscope/auto';

    import { AppRegistry } from 'react-native';
    import App from './App';

    AppRegistry.registerComponent('main', () => App);
    ```
  </TabItem>
  <TabItem label="Flutter">
    ```dart title="lib/main.dart"
    import 'package:owlscope/auto.dart';
    import 'package:flutter/material.dart';

    void main() {
      owlscopeAuto(() => runApp(const MyApp()));
    }
    ```
  </TabItem>
  <TabItem label="Node.js">
    ```ts title="server.ts"
    import 'owlscope/auto';

    import express from 'express';
    const app = express();
    app.listen(3000);
    ```
  </TabItem>
</Tabs>

Open the desktop app, run your project — events stream in.
